# üîß How to run

```bash
# Clone Repository
$ git clone git@github.com:marcoaminotto/ui-clone-typeform-homepage.git
```

### üíª Run web application

```bash
# Go to web folder
$ cd ui-clone-typeform-homepage/web

# Install Dependencies
$ yarn install

# Run Aplication
$ yarn start
```

Go to http://localhost:3000/ to see the nice animation.
